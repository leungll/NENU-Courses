G.711 Test Data
_____ ____ ____


File:			voice.pcm
Description:	pulse code modulation sound file
			8kHz, mono, 3.204 seconds
Data format:	binary
Data size:		16-bit
Endian:		little

File:			voiceA.wav
Description:	A-law sound file
			8kHz, mono, 3.204 seconds
Data format:	binary
Data size:		8-bit
Endian:		little

File:			voiceU.wav
Description:	u-law ("mew"-law) sound file
			8kHz, mono, 3.204 seconds
Data format:	binary
Data size:		8-bit
Endian:		little

Application:	SoX (Sound eXchange)
Creator:		Lance Norskog
